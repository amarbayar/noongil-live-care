# Public Repo Scope

This repository is a curated public submission slice prepared for hackathon judging.

## Included

- iOS app source for the live-agent experience
- Backend source, tests, and deployment assets
- Shared contracts and config files needed to understand the implementation
- Local model assets and frameworks required to build the demoable app
- Legal docs referenced by the app bundle

## Excluded

- Private strategy, roadmap, and internal planning docs
- Local environment files and secrets
- Original private repo metadata
- Build artifacts and local dependency caches
- Legacy prototype folders not required for the submission

## Why

The goal is to keep the public repo:

- inspectable by judges
- reproducible enough for technical review
- aligned with the actual demo
- free of unnecessary confidential business material

## Known Limits

- Some app and backend defaults are placeholders and must be replaced with your own project settings
- Some production-hardening work remains intentionally out of scope for the hackathon slice
